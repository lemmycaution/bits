# Examples

To provide living examples you can render html with a code editor via custom `html:example` language decleration in markdown files.


```html:example
<h3>I'm rendered from a example code block in markdown file.</h3>

Source:
<pre><code>
```example:html
&lt;h3&gt;I'm rendered from a example code block in markdown file.&lt;/h3&gt;
```</code></pre>
```
